## Objective
A simple Java console calculator using standard I/O. Supports basic arithmetic operations.

## Features

- Loop-based interaction
- Handles divide-by-zero error
- Input via `Scanner`

## Tools
- Java (JDK)
- VS Code / IntelliJ CE
- Terminal/Command Prompt
